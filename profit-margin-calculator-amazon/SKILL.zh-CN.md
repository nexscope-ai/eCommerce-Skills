# Amazon Profit Calculator (Lite)

亚马逊利润计算器 - 对话版

帮助卖家计算产品成本明细、利润率、盈亏平衡点，并给出定价建议。

## 适用场景

- 新品上架前利润预估
- 现有产品利润诊断
- 定价策略调整
- 批量 SKU 利润分析

---

## 使用流程

### Step 1: 触发 Skill

```
帮我算一下这个产品能不能赚钱
计算利润
利润分析
```

### Step 2: 产品信息收集

引导用户提供：

```
📦 产品基本信息

请告诉我以下信息：

• 产品名称/SKU: ___
• 售价: $___
• 品类: ___ (如 kitchen, electronics, toys)
```

### Step 3: 成本数据收集

```
💰 成本数据

**必填**
• 产品成本 (FOB): $___/件
• 头程运费: $___/件
• FBA 配送费: $___/件 (不知道可以估算)
• FBA 仓储费: $___/件/月

**可选**
• 广告费占比: ___% (默认 10%)
• 退货率: ___% (默认 3%)
• 其他杂费: $___/件

直接用自然语言回复，如：
"产品成本6块，头程1.5，FBA费5.5，仓储0.3，广告10%"
```

### Step 4: 分析输出

```
📊 输出内容

1. 成本明细表
2. 利润率 (毛利/净利)
3. 盈亏平衡分析
4. 定价建议 (可自定义目标利润率)
```

### Step 5: 定价建议

```
💡 需要调整目标利润率吗？

默认给出 15%/20%/25%/30% 四档建议。
如果你有特定目标，直接告诉我，如 "我想要 35% 利润率"
```

## 成本项目说明

| 成本项 | 说明 | 计算方式 |
|--------|------|---------|
| 产品成本 | FOB 采购价 | 用户输入 |
| 头程运费 | 海运/空运到 FBA | 用户输入 |
| FBA 配送费 | 亚马逊仓储配送费 | 用户输入 |
| FBA 仓储费 | 月度仓储费 | 用户输入 |
| Referral Fee | 平台佣金 (8-15%) | 自动 × 品类费率 |
| 广告费 | PPC 花费 | 售价 × 广告占比 |
| 退货成本 | 退货处理损失 | 退货率 × 处理费 |
| 其他杂费 | 包装/标签等 | 用户输入 |

## 批量计算

### CSV 格式

```csv
sku,name,selling_price,product_cost,shipping_cost,fba_fee,storage_fee,ad_ratio,category
A001,Product A,29.99,6,1.5,5.5,0.3,0.1,kitchen
A002,Product B,19.99,8,1,4.5,0.2,0.12,home
```

### 批量输入方式

```
帮我批量计算利润:
A001, 产品A, 售价29.99, 成本6, 头程1.5, FBA 5.5
A002, 产品B, 售价19.99, 成本8, 头程1, FBA 4.5
```

## 利润状态判定

| 净利率 | 状态 | 建议 |
|--------|------|------|
| >20% | ✅ 健康 | 保持 |
| 5-20% | ⚠️ 警告 | 优化成本或提价 |
| 0-5% | 🔴 危险 | 需要调整 |
| <0% | 💀 亏损 | 停止或大幅调整 |

## 脚本使用

```bash
# 单品分析 (默认测试数据)
python3 scripts/calculator.py

# 自定义输入
python3 scripts/calculator.py '{"selling_price": 29.99, "product_cost": 6, ...}'

# 批量分析
python3 scripts/calculator.py '[{...}, {...}]'
```
